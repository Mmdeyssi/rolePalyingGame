## 🎮 What is this project?

A browser-based **Role Playing Game** built entirely with vanilla JavaScript — no frameworks, no libraries. This project demonstrates core JavaScript skills including DOM manipulation, game state management, event handling, and interactive UI logic.

---

## ✨ Features

- 🧙 Interactive RPG gameplay in the browser
- ⚔️ Game state management with pure JavaScript
- 🎨 Custom CSS styling and animations
- 📱 Runs directly in any modern browser
- ☁️ Deployed live on Vercel

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Game structure and UI |
| CSS3 | Styling and visual effects |
| JavaScript (ES6+) | Game logic and DOM manipulation |
| Vercel | Deployment |
